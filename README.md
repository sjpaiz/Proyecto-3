**Simulador de Máquina de Turing**
Este proyecto utiliza Python y Tkinter.  
Permite seleccionar entre distintas máquinas, ingresar una cadena y observar el funcionamiento del cabezal paso a paso o de forma automática.

**Instalaciones necesarias**
No se requiere instalación adicional.  
El programa utiliza únicamente librerías estándar incluidas con Python.
